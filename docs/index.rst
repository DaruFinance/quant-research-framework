quant-research-framework
========================

Walk-forward backtester with statistical validation, robustness testing,
and a tolerance-bounded cross-language parity contract against
`quant-research-framework-rs`_.

.. _quant-research-framework-rs: https://github.com/DaruFinance/quant-research-framework-rs

This is the Python reference implementation. The companion paper
introducing the framework and its parity discipline is *A
Reproducibility-First Walk-Forward Backtester with Tolerance-Bounded
Cross-Language Parity* (Vieira Gatto, 2026).
